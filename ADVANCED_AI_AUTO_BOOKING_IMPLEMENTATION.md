#### 🧠 RAG 시스템 핵심 기능:

##### ✅ **문서 기반 질의응답 (RAG)**
```typescript
// 지식 베이스 통합
```

##### ✅ **부킹 히스토리 학습**
```typescript

- 선호 항로 자동 감지
- 평균 부킹 규모 계산
- 부킹 빈도 분석
- 비용 절약 추적
```

##### ✅ **개인화된 추천**
```typescript
// 맞춤형 인사이트
- 사용자별 최적 부킹 시점
- 개인화된 항로 추천
- 맞춤형 절약 전략
- 행동 패턴 기반 제안
```

##### ✅ **다국어 자연어 처리**
```typescript
// 고급 언어 지원
- 한국어/영어 완벽 지원
- 컨텍스트 기반 번역
- 문화적 뉘앙스 반영
- 전문 용어 정확성
```

#### 🎯 RAG 시스템 아키텍처:

```typescript
사용자 질문
    ↓
문서 검색 (Vector Similarity)
    ↓
관련 문서 추출 (Top-K)
    ↓
사용자 컨텍스트 통합
    ↓
AI 모델 (Gemini/OpenRouter)
    ↓
개인화된 응답 생성
    ↓
신뢰도 점수 계산
    ↓
소스 출처 표시
```

#### 📊 향상된 UI/UX:
- **RAG 모드 토글**: 지식 베이스 활용 ON/OFF
- **신뢰도 표시**: AI 응답의 확신도 시각화
- **소스 출처**: 참조한 문서 자동 표시
- **개인화 정보**: 사용자 패턴 실시간 분석
