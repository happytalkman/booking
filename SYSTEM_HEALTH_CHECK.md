## ⚠️ 주의사항

### 1. Python 감정 인식 서버
**상태**: 설치 필요

**설치 방법**:
```bash
cd server
pip install -r requirements.txt
python emotion_detection.py
```

**폴백**: 서버 미실행 시 밝기 기반 분석으로 자동 전환

### 2. API 키 설정
**필수 설정**: `.env.local`
```
VITE_OPENROUTER_API_KEY=your_key
VITE_GEMINI_API_KEY=your_key
```
