## 🔍 추천 시스템 상세 분석

### 1. **AI 부킹 추천** (BookingRecommendation.tsx)
**기능**: 운임 예측 기반 부킹 타이밍 추천
```typescript
추천 유형:
- 지금 부킹: 운임 상승 예상 시 (신뢰도 92%)
- 대기 권장: 운임 하락 예상 시 (신뢰도 78%)
- 모니터링: 불확실한 상황 (신뢰도 65%)

예상 효과:
- 평균 절감: $200/FEU
- 정확도: 85%
```

### 2. **온톨로지 AI 추천** (OntologyRecommendationEngine.tsx)
**기능**: 지식 그래프 구조 개선 추천
```typescript
추천 유형:
- 누락된 관계: 삼성전자 → 부산-뉴욕 항로
- 새로운 엔티티: 부산항, 리스크 이벤트
- 구조 최적화: 계약 노드 → 관계 속성
- 데이터 품질: 중복 노드 병합

신뢰도: 75-98%
```

### 3. **멀티모달 AI 추천** (MultimodalAIService)
**기능**: 음성/이미지/텍스트 분석 후 액션 추천
```typescript
추천 유형:
- 부킹 액션: 음성 명령 기반
- 문서 분석: 선하증권 자동 처리
- 컨텍스트 액션: 상황별 최적 작업

신뢰도: 80-95%
```

### 4. **딥러닝 예측 추천** (DeepLearningPredictionEngine)
**기능**: LSTM+Transformer 기반 고정확도 추천
```typescript
추천 유형:
- 트렌드 기반: 상승 시 조기 부킹
- 리스크 기반: 불확실성 높을 때 보수적 접근
- 정확도 기반: 신뢰도에 따른 전략 수립

신뢰도: 94.7% (업계 최고)
```

### 5. **보안 추천** (SecurityService)
**기능**: 시스템 보안 강화 추천
```typescript
추천 유형:
- API 키 설정: 실제 키로 교체
- HTTPS 활성화: 프로덕션 보안
- 프로덕션 빌드: 개발 → 운영 전환

신뢰도: 95-100%
```

### 6. **개인화 추천** (RAGService)
**기능**: 사용자 맞춤형 추천
```typescript
추천 유형:
- 선호 항로: 자주 사용하는 항로 최적 타이밍
- 부킹 패턴: 개인 히스토리 기반 개선
- 학습 기반: 사용 패턴 분석 결과

신뢰도: 80-85%
```

### 7. **리포트 추천** (ReportService)
**기능**: 분석 리포트 내 권장사항
```typescript
추천 유형:
- 환율 헤지 전략
- 유류할증료 조정
- 운임 조정 권장

신뢰도: 85-90%
```

### 8. **🆕 스마트 통합 추천** (SmartRecommendationCenter)
**기능**: 모든 추천을 통합하고 개인화
```typescript
추천 유형:
- 부킹 최적화: 선호 항로 기반 타이밍
- 항로 발견: 새로운 경제적 대안
- 계절 전략: 비수기/성수기 최적화
- 비용 절감: 컨테이너 통합 등
- 리스크 분산: 단일 항로 의존도 감소
- 기능 활용: 미사용 기능 안내
- 패턴 학습: 사용 데이터 기반 개선

특징:
- ML 기반 개인화
- 우선순위 자동 정렬
- 피드백 학습 시스템
- 실시간 유효성 검증
```
