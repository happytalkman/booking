## 🔍 발견된 주요 이슈

### 1. 보안 결함 (Critical)

#### 1.1 API 키 노출 위험
- **위치**: `.env.local`, `vite.config.ts`
- **문제**: 
  - GEMINI_API_KEY가 PLACEHOLDER로 설정됨
  - OpenRouter API 키가 하드코딩됨
  - 로컬호스트 URL이 하드코딩됨
- **위험도**: 높음

#### 1.2 토큰 보안 취약점
- **위치**: `services/authService.ts`
- **문제**: 
  - JWT 토큰이 Base64로만 인코딩 (암호화 없음)
  - 토큰 만료 검증 로직 부족
  - localStorage에 민감 정보 저장
- **위험도**: 높음

#### 1.3 개발용 코드 프로덕션 포함
- **위치**: `test-*.html`, `test-*.js`
- **문제**: 테스트 파일들이 프로덕션에 포함됨
- **위험도**: 중간

### 2. 미개발/미완성 기능

#### 2.1 API 키 미설정
- **GEMINI_API_KEY**: PLACEHOLDER 상태
- **실제 외부 API**: 대부분 DEMO 키 사용
- **영향**: AI 기능 및 실시간 데이터 기능 제한

#### 2.2 백엔드 의존성
- **인증 시스템**: localhost:3001 의존
- **이메일 서비스**: 설정 필요
- **실제 데이터 API**: 대부분 목업 데이터 사용

### 3. 톤앤매너 불일치

#### 3.1 언어 혼용
- **문제**: 영어 컴포넌트 내 한국어 하드코딩
- **위치**: 다수 컴포넌트의 className, title, placeholder
- **예시**: `title: '실시간 시장 분석'` (영어 컴포넌트 내)

#### 3.2 일관성 부족
- **메시지 톤**: 일부는 격식체, 일부는 반말
- **용어 통일**: 같은 기능을 다른 용어로 표현
