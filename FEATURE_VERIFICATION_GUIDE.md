# 고급 기능 확인 가이드

## 🧠 AI 예측 정확도 향상 기능

### 확인 방법:
1. **브라우저에서 http://localhost:3001 접속**
2. **우하단 뇌 모양 아이콘 클릭** (AI Learning Dashboard)
3. **확인할 수 있는 내용:**
   - 📊 **온라인 학습 메트릭**: 실시간 학습 진행 상황
   - 🏆 **모델 성능 비교**: 여러 AI 모델의 정확도 비교
   - 📅 **계절성 패턴**: 월별 계절 요인 분석
   - ⭐ **최고 성능 모델**: 현재 가장 좋은 모델 표시

### 실시간 업데이트:
- 30초마다 자동으로 데이터 갱신
- 모델 정확도가 90% 이상이면 초록색, 80% 이상이면 파란색 아이콘

## 📊 고급 데이터 시각화 기능

### 확인 방법:
1. **브라우저에서 http://localhost:3001 접속**
2. **화면 아무 곳이나 클릭** (Advanced Visualization Dashboard 활성화)
3. **4가지 시각화 모드 확인:**

#### 📈 실시간 차트
- 환율 (USD/KRW, EUR/KRW)
- 유가 (Brent, WTI)
- 날씨 데이터 (온도, 폭풍 위험도)
- 지정학적 리스크

#### 🗺️ 리스크 히트맵
- 아시아, 유럽, 중동, 아메리카 지역별 위험도
- 색상으로 위험 수준 표시 (녹색=낮음, 빨간색=높음)

#### ⏰ 타임라인
- 시간대별 주요 이벤트
- 데이터 변화 추이

#### 🎯 3D 뷰
- 3차원 데이터 시각화
- 인터랙티브 3D 차트

### 실시간 업데이트:
- 5초마다 자동 데이터 갱신
- 자동 새로고침 ON/OFF 가능

## 📄 스마트 리포트 생성기 기능

### 확인 방법:
1. **브라우저에서 http://localhost:3001 접속**
2. **우하단 문서 아이콘 클릭** (Smart Report Generator)
3. **3개 탭에서 기능 확인:**

#### 📋 생성된 리포트 탭
- 이미 생성된 리포트 목록
- 다운로드 버튼으로 HTML 파일 다운로드
- 리포트 크기, 생성일시 정보

#### 📝 템플릿 탭
- **일일 리포트**: 매일 자동 생성
- **주간 리포트**: 매주 자동 생성  
- **월간 리포트**: 매월 자동 생성
- **사용자 정의**: 맞춤 리포트

#### ⏰ 스케줄 탭
- 자동 리포트 생성 일정 설정
- 이메일 발송 설정
- 생성 주기 관리

### 리포트 생성 테스트:
1. **템플릿 선택** (예: 일일 리포트)
2. **"지금 생성" 버튼 클릭**
3. **생성 완료 후 다운로드**

## 🔔 실시간 알림 시스템 (추가)

### 확인 방법:
1. **브라우저에서 http://localhost:3001 접속**
2. **상단 헤더의 알림 아이콘 확인**
3. **알림 허용 팝업에서 "허용" 클릭**
4. **실시간 알림 받기:**
   - 환율 급변동 (±2% 이상)
   - 유가 급등락 (±3% 이상)
   - 날씨 위험 (폭풍 위험도 70% 이상)
   - 지정학적 리스크 증가

---

## 🚀 전체 시스템 상태 확인

### 개발자 도구에서 확인:
1. **F12 키로 개발자 도구 열기**
2. **Console 탭에서 로그 확인:**
   ```
   ✅ AI Learning Service initialized
   ✅ Real-time alerts enabled
   ✅ Advanced visualization ready
   ✅ Smart report generator active
   ```

### API 연동 상태 확인:
- **우하단 API 상태 표시기** 확인
- 초록색: 모든 API 정상
- 노란색: 일부 API 지연
- 빨간색: API 오류
